# Amazon Live Coding Mock

A tiny standalone HTML mock interview environment meant to feel similar to a typical live coding screen:

- problem statement panel
- timer
- code editor area
- sample test/output panel

## Open it

From this directory, open `index.html` in your browser.

Examples:

### macOS
```bash
open index.html
```

### Linux
```bash
xdg-open index.html
```

## Notes

- This is a mock UI only.
- It does **not** execute arbitrary code.
- It is intended to simulate the feel of an interview environment so you can practice:
  - explaining your approach
  - coding in a plain editor
  - testing edge cases aloud
  - working with limited IDE help

## Files

- `index.html` — standalone mock page with embedded CSS and JS
